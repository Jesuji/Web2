API 명세서

기능 엔드포인트 HTTP 메서드 설명
회원가입 /auth/sign POST 새로운 사용자 등록
로그인 /auth/login POST 사용자 로그인
로그아웃 /auth/logout POST 사용자 로그아웃
프로필 조회 /my-profile GET 사용자 프로필 조회 (사진 포함 가능)
마커 띄우기 /restaurants/search/location POST 지도에 음식점 마커 표시
음식점 세부정보 조회 /restaurants/{id} GET 선택한 음식점의 상세 정보 조회
음식점 리뷰 조회 /restaurants/{id}/reviews GET 선택한 음식점의 리뷰 목록 조회
음식점 리뷰 작성 /reviews/new?restaurantId=343603 POST 음식점 리뷰 작성 (이미지 포함 가능)
리뷰 수정 /reviews/update/{id} PATCH 작성한 리뷰 수정
리뷰 삭제 /reviews/delete/{id} DELETE 작성한 리뷰 삭제
키워드 검색 /restaurants/search?keyword=메뉴이름 GET 음식점 검색
